统一 skill 的三步法
1

先统一人物和世界观

阿鱼、欢迎语、头像、树叶纹理、护照卡这些元素必须在各页型里反复出现。

2

再统一组件和配色

所有页面都围绕暖米卡片、薄荷高亮、阳光黄贴纸和蜜柑橙点缀，不再混入旧蓝黄红系统。

3

最后统一语气和生活感

每个页面都像岛上正在被使用的工具、公告板或图鉴，而不是演示用的正式网页。

Memo 01 · 不要回到“教程网站感”

就算内容结构是教程，也要让页面像贴在岛上布告栏的一组认真整理过的小纸条。

Memo 02 · 教程也要有角色感

这里不是匿名知识库，而是阿鱼的小岛经验分享，所以身份卡和欢迎语都值得保留。

MODULE A

适合什么内容

方法拆解、流程指南、收集攻略、品牌规范说明，都适合走这类教程页语言。

MODULE B

看起来应该像什么

像温暖的知识布告板，不像在线课程后台，不像文档产品，不像企业培训页。
