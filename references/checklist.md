# 质量检查清单

> 做完以后逐条过。P0 任何一条不过，都不能算完成。

---

## P0

- [ ] 整体气质符合 Ayu 的 `brand-dna.md`，不是 Esther 旧风格，不是通用 AI 模板
- [ ] 页面外层与内容层有明确区分：外层可为森林绿叶片纹理，内容层必须更暖更浅
- [ ] 至少出现一个“小岛原生系统组件”，不是纯网页 section
- [ ] 没有 HTML 默认样式裸奔
- [ ] 没有蓝紫梦幻渐变、冷灰背景、玻璃拟态、科技感 neon
- [ ] 页面截图看起来像“生活中的系统界面”，不是 SaaS、不是博客模板
- [ ] 桌面和移动端都能成立

---

## P1

- [ ] 页面里能看到角色存在感、欢迎感或居住感
- [ ] 布局不是整页同一种白卡网格
- [ ] 组件之间有家族相似性，也有职责差异
- [ ] 图标、贴纸、标签、印章等细节和主风格一致
- [ ] 长内容被拆成可阅读的模块，而不是大段文字堆叠

---

## P2

- [ ] 有一处让人想截图保存的主视觉卡面
- [ ] 有至少一个“系统页语义”明显的模块，比如图鉴、护照、任务板、状态板
- [ ] 页面整体带一点生活气和温度，不只是整洁
