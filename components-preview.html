ANIMAL-ISLAND-UI BASED

阿鱼的 UI 组件总览页

这页不再沿用 Esther 那种杂志式 51 组件目录，而是基于 `animal-island-ui` 的 NookPhone 语汇来组织组件。重点不是“组件数量很多”，而是这套组件能不能自然组成阿鱼的小岛系统。

📱 AYU UI KIT
小岛组件手机
用真实的 Animal Island icon 和 NookPhone 结构来展示这套 UI 的核心组件。

图鉴入口卡

适合知识分类、条目入口、收藏目录，是这套系统最像游戏原生的导航组件。

设计入口卡

适合页面类型、视觉模块、玩法说明等更偏功能入口的内容。

进度状态卡

适合任务、收藏、完成度、阶段状态，让系统更像有收集机制的小岛工具。

为什么 UI 总览要改成这样

因为你现在确认的整套视觉核心，不是“通用网页组件库”，而是“阿鱼的小岛系统组件”。所以总览方式也应该像手机内置 App 图鉴，而不是大而全的组件样品册。
